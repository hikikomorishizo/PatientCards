\# PatientCards



An Android application for managing patient records.



\## Project Goal

The goal of this project is to create a convenient tool for working with patient data, including:

\- Creating and editing patient cards

\- Fast search by name and date

\- Viewing patient visits by calendar dates

\- Integration with Google Calendar

\- Support for custom abbreviations and text expansion



\## Planned Features

\- Patient database stored locally on the device

\- Visit history linked to specific dates

\- Clickable calendar events that open patient records

\- User-defined abbreviations with suggestions or auto-expansion

\- Clean and intuitive UI



\## Tech Stack

\- \*\*Kotlin\*\*

\- \*\*Jetpack Compose\*\*

\- \*\*MVVM architecture\*\*

\- \*\*Room (SQLite)\*\*

\- \*\*Google Calendar API\*\*
